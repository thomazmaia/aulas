# SQLAlchemy

O SQLAlchemy é uma biblioteca que transforma comandos SQL em código Python. Com ela é possível criar classes (em Python) que representam tabelas no banco de dados e trabalho com objetos ao invés de escrever SQL diretamente.

### Instalação
```pip install flask-sqlalchemy```

### Estrutura de pastas:
```
app/
--- controllers/
------- (rotas)
--- models/
------- (classes python)
--- static/
------- (CSS e JS)
--- templates/
------- (HTML)
```
