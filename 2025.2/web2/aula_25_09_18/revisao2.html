D.O.M - Document Object Model

D.O.M é a estrutura organizacional utilizada pelo navegador web (Google Chrome) para representar uma página HTML. Essa estrutura carrega todos os elementos da página e pode ser alterada através do JavaScript (JS). Em outras palavras, é através dele que alteramos uma página web.

1. Como ACESSAR elementos

  1. getElementById() - Seleciona um elemento da página através do seu ID (único!). O retorno dessa função é um objeto que é o próprio elemento.
  2. getElementsByClassName() - Seleciona todos os elemento da página que contenham a mesma classe. O retorno dessa função é um array/lista de objetos equivalentes aos elementos da página web.
  3. querySelector() - Seleciona o primeiro elemento dado o id/classe fornecido.
  4. querySelectorAll() - Seleciona todos os elementos dado o id/classe fornecido. O retorno dessa função é semelhante ao getElementsByClassName (um array de elementos).

2. Como MANIPULAR elementos

Depois de selecionar o objeto desejado, você consegue acessar suas propriedades. Cada elemento tem propriedades únicas.

Ex:

var obj_txt = document.querySelector(".txt");
obj_txt.textContent = "Novo texto";

3. Como acessar e manipular ATRIBUTOS

Assim como é possível manipular elementos, também é possível manipular seus atributos.

Ex:

var obj_img = document.querySelector("#img");
obj_img.setAttribute("src", "nova_imagem.jpg");

4. Como acessar e manipular ESTILOS (CSS)

Como acessar e manipular seletores de estilos de CSS através do JavaScript.

Ex de CSS:

#caixa {
  width: 100px; 
  height: 100px;
  background: red;
}

Ex de JS:

var caixa = getElementById("caixa");
caixa.style.width = "200px";
caixa.style.backgroundColor = "blue";

5. Como criar elementos

5.1. document.write()

É possível "escrever" textos na página utilizando o document.write(). Contudo, ele sempre "apagará" a página atual e escreverá em uma nova.

Ex:

document.write("<h1>Esse é um título</h1>")
document.write("<p>Esse é um texto</p>")

5.2. createElement()

É possível criar e adicionar novos elementos (ou estruturas) dinamicamente através do JS.

Ex:

var paragrafo = document.createElement('p');
paragrafo.textContent = "Esse é um texto";
var imagem = document.createElement('img');
imagem.setAttribute("src", "caminho_da_imagem.png");
