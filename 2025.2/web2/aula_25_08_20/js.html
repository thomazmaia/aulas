JavaScript - JS

1. História

O JavaScript (JS) nasceu como uma liguagem "quebra galho" para evitar várias solicitações a servidores numa época em que a internet era muito lenta.

O JS roda/executa em browser/navegador (ex: Google Chrome). Nasceu com o nome de "Mocha" mas logo foi renomeado para JavaScript para "surfar" a onda do Java, que era uma linguagem que estava nascendo e sendo muito bem aceita. Um tempo depois foi padronizada como ECMAScript.

Apesar de ter nascido como linguagem própria para FrontEnd, hoje é possível utilizá-la como linguagem de BackEnd através do NodeJs (frameworks em JS para BackEnd).

2. Estrutura

  • index.html
  • CSS/style.css
  • JS/script.js

Arquivo: ex_JS1.html

<html>
  <head>
    <meta charset="UTF8">
    <title> Olá Mundo </title>
    <script>
        alert("Olá, Mundo!");
        console.log("Olá, Mundo!");
    </script>
  </head>

  <body>
  </body>

</html>

Arquivo: ex_JS2.html

<html>
  <head>
    <meta charset="UTF8">
    <title> Olá Mundo </title>
    <script src="JS/script.js"></script>
  </head>

  <body>
  </body>

</html>

Arquivo: script.js (dentro da pasta "JS")

alert("Olá, Mundo!");
console.log("Olá, Mundo!");
