# Lista de exercícios 3 - WHILE

1.  Crie um código em Python que imprima os números de 2 a 20 usando o
    comando "while" e uma variável contadora para controlar o loop.

2.  Crie um código em Python que peça ao usuário para digitar um número
    inteiro positivo e então utilize o comando "while" para imprimir
    todos os números pares até chegar ao número digitado pelo usuário.

3.  Crie um código em Python que use o comando "while" para ler números
    inteiros inseridos pelo usuário até que o número inserido seja -1 e
    imprima a média de todos os números inseridos (desconsidere o -1 no
    cálculo).

4.  Crie um código em Python que solicite ao usuário um número e, usando
    o comando "while", imprima todos os múltiplos desse número menores
    que 100. O programa deve continuar solicitando novos números até que
    o usuário digite 0 para encerrar.

5.  Leia um número N e calcule o fatorial de N usando while.\
    <br> OBS: **NÃO** utilize funções prontas como factorial.
    Faça o cálculo manualmente.

6.  Gere um número aleatório entre 10 e 50 (incluindo 10 e 50). Peça ao
    usuário para adivinhar o número e informe se o palpite é menor ou
    maior que o número secreto. O jogo deve continuar até que o usuário
    acerte.\
    <br> OBS: `import random` e `num = random.randint(10,50)`

7.  Escreva um programa para escrever todos os números de 1 a 50. Onde
    tiver um múltiplo de 4, troque por "Quad". Onde tiver um múltiplo de
    6, troque por "Hex". Para os múltiplos de 4 e 6 ao mesmo tempo,
    troque por "QuadHex".

8.  Crie um programa para converter de decimal para binário usando
    while.

9.  Um número perfeito é aquele cuja soma de seus divisores próprios é
    igual a ele mesmo. Crie um programa para verificar se um número N é
    um número perfeito.\
    <br> Ex: 6 = 1 + 2 + 3

10. Modifique a questão anterior para encontrar todos os números
    perfeitos dentro de um intervalo informado pelo usuário.
