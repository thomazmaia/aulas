# Lista de Exercícios 1 - VARIÁVEIS E OPERADORES

1. O volume de um cilindro com raio `r` e altura `h` é dado por $\pi r^2 h$. Qual é o volume de um cilindro com raio `r` (fornecido pelo usuário)  e altura `h` (também fornecida pelo usuário)? <br>OBS: inicie as variáveis `r` com valor 4 e `h` com valor 10.

2. Suponha que o preço de capa de um livro seja R$ 32,90, mas as livrarias recebem um desconto de 35%. O transporte custa R$ 4,00 para o primeiro exemplar e 0,80 centavos para cada exemplar adicional. Qual é o custo total de atacado para 75 cópias.

3. Você olha para um relógio e são exatamente 9 da manhã. Você coloca um alarme para tocar daqui a 37 horas. A que horas o alarme irá tocar? <br>OBS: desconsidere minutos e segundos

4. Escreva um programa em Python que resolve a versão geral do problema acima. Peça ao usuário que entre com a hora atual em horas e que entre com o número de horas que deverá esperar antes do alarme tocar. Seu programa deve imprimir a hora que o alarme irá tocar.

5. Escreva um programa para ler um número `N` e mostre `N`, `NN` e `NNNN`. <br>OBS: **NÃO** converta `N` para string. Trate-o como número! <br>Ex: para N = 3, mostre: 3, 33, 3333

6. Crie um programa para calcular a área de um trapézio dado o valor da base maior, da base menor e da altura.

7. Escreva um programa para resolver $(a - b) * (a - b)$.

8. Escreva um programa para converter distância de quilômetros para metros, centímetros e milhas.

9. Crie um programa para calcular a distância entre os pontos $(x1, y1)$ e $(x2, y2)$.

10. Escreva um programa para calcular a soma dos dígitos de um número de 5 dígitos. <br>Ex: para N = 73845, mostre: 27
