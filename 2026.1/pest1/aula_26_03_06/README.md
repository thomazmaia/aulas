# Lista de Exercícios 4 - FOR

1.  Crie um código em Python que use um `for` loop para imprimir todos
    os números ímpares de 1 a 25 (inclusive).

2.  Escreva um código em Python que utilize um laço `for` para imprimir
    os números de 1 a 20 (inclusive), mas pulando os números 4, 9 e 17.

3.  Crie um programa em Python que imprima a soma dos números múltiplos
    de 3 de 0 até um número inteiro dado pelo usuário. Utilize o comando
    `for` para percorrer os números e realizar a soma.

4.  Crie um código em Python que use o comando `for` para percorrer de 1
    até n (inclusive) números e calcule o produto acumulado desses
    valores.

5.  Crie um programa para verificar se um número é perfeito ou não.
    Utilize o comando `for`.

6.  Escreva um código em Python que utilize o laço `for` para calcular a
    soma dos cubos dos números inteiros de 1 a 50 (inclusive) e
    armazená-la em uma variável chamada 'soma'. Utilize uma única linha
    de código dentro do laço `for`.

7.  Crie um código em Python que calcule a quantidade de números primos
    de 1 a 200 (inclusive) usando o comando `for`.

8.  Crie um código em Python que use o comando `for` para percorrer de 1
    até n (inclusive) números e calcule o quadrado de cada número,
    exibindo o resultado na tela.

9.  Um número é chamado de número narcisista se for igual à soma de seus
    dígitos elevados à quantidade de dígitos. Crie um programa usando
    `for` para encontrar todos os números narcisistas dentro de um
    intervalo informado pelo usuário.
    
    <br>Ex: 153 = 1³ + 5³ + 3³

10. Crie um código em Python que use o comando `for` para calcular o
    valor da somatória dos números inteiros positivos menores que n que
    são divisíveis por 4 ou 6. Onde n é um número inteiro inserido pelo
    usuário.
